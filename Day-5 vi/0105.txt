명령어
-cat (conCATenate) : 파일의 내용 확인

cat -n : 라인 번호를 포함하여 출력
cat -b : 라인 번호를 출력, 공백은 제외
cat > [파일명] : 파일에 입력할 수 있도록 되어있음
-------------------------------------------------------------------------------------
-more : 화면단위로 파일 내용 출력, cat 명령어는 너무 많은 양이 슉 나와버리지만
	more 은 스페이스바를 누를때마다, 엔터키를 누를때마다 화면과 줄이 넘어감
	/[pattern] : 문자열 검색 됨
more [파일명]
-------------------------------------------------------------------------------------
-less : 개선된 화면 단위 파일 내용 출력
	내용을 앞뒤로 스크롤 이동할 수 있다.
	Ctrl + f : 다음화면(=space bar)
	Ctrl + b : 이전화면
-------------------------------------------------------------------------------------
-head : 파일 내용의 첫 10줄만 출력
head -n [번호] [파일명] : 지정한 번호만큼 위에서부터 출력


-tail : 파일 내용의 마지막 10줄만 출력

tail -n [번호] [파일명] : 지정한 번호만큼 아래서부터 출력
★tail -f : 실시간으로 파일의 내용을 확인할 수 있다.★

--------------------------------------------------------------------------------------
-touch : 빈파일 생성, 파일 접근 시간 변경

touch -t YYYYMMDDhhmm [파일명] : 생성시간 설정, 과거의 시간으로 설정가능

ex) touch [파일명]
------------------------------------------------------------------------------------
-file : 파일의 형식을 확인

ex) file [파일명]   file /etc/passwd
---------------------------------------------------------------------------------------type : 파일 위치, bash 빌트인 명령어 여부, alias 정보를 확인
type [파일명]
--------------------------------------------------------------------------------------
-whereis : 명령어에 대한 바이너리, 소스, 메뉴얼 페이지 위치를 확인
whereis [파일명]

ex) #whereis ls -> ls: /bin/ls /usr/share/man/man1/ls.1.gz
-------------------------------------------------------------------------------------
-which : 실행 파일의 경로 및 alias 정보 확인

-------------------------------------------------------------------------------------
-grep (Grobally find Regular-Exprssion and Print) :문자열 검색
grep [옵션] [패턴] [파일명]

grep -i : 대소문자 구별하지 않는다
grep -n : 행번호를 함께 보여준다
grep -v : 패턴에 일치하는 단어는 제외하고 출력
grep -w : 패턴이 전체 단어와 일치하는 줄만 출력

$$$그런데!$$$$

파일을 이렇게 만들었을 때
abc : abc
abc1:abcc
abc2:abccc

grep abc abc*->
abc:abc
abc1:abcc
abc2:abccc

grep abc* abc*->
abc1:abcc
abc2:abccc
abc:abc
abc1:abcc
abc2:abccc

왜냐면 abc*=abc abc1 abc2 라는 뜻이고 grep 에서 패턴은 맨처음 abc 만 패턴이고 나머지는 찾을 파일로 인식한다.
따라서 패턴에 *을 붙여야할 경우 #grep 'abc*' abc* 라고 입력하면 된다.
